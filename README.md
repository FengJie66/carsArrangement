# carsArrangement
汽车信息管理系统-前端

基于springboot+mybatisplus+vue+mqtt协议的前后端分离汽车信息管理系统


npm install         // 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn

// 运行
npm run dev

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build
